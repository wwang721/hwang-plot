⏺ The root cause was that the TCAN models provide raw residuals instead of
  pre-binned histogram data, and use theoretical_q/sample_q instead of
  theoretical_quantiles/sample_quantiles (with no qq_line).

  The fix handles both cases:
  - Histogram: detects bin_left column → pre-binned path; otherwise treats the
  single column as raw residuals and bins them with np.histogram(bins='auto')
  - QQ plot: auto-detects column names; if no qq_line is provided, computes the
  reference line from the interquartile range (25th–75th percentile fit)
